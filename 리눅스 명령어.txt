0. history - 사용한 모든 명령어 보여주기

1. man [명령어]: 명령어 사용법 확인
2. pwd: 현재 디렉토리 위치 확인

3. ls: 목록 보기
	-a: 숨김파일이나 폴더까지 모두 출력
	-l : 목록형식(세로방향) 출력
	-h: 크기를 바이트에서 M 단위로 변경해서 출력
	-S: 크기의 내림차순 출력

4. cd: 디렉토리 변경
	ex) cd /d/naver : d 드라이브의 naver폴더로 이동
	cd ~: root(맨 위)
	cd - : 직전 디렉토리로 이동

5. mkdir : 디렉토리 생성
	-p: 하위 디렉토리까지 같이 생성
	mkdir hello - hello 디렉토리 생성
	mkdir -p bitcamp/member/study	- 하위디렉토리 포함 한꺼번에 경로대로 생성

6. rmdir : 빈 디렉토리 삭제 - 빈 디렉토리가 아니면 에러 발생
7. rm : 파일or디렉토리 삭제
	-r : 디렉토리 삭제 시 하위 같이 삭제
	-f: 삭제 물음 없이 강제 삭제

8. cat : 파일 내의 내용 확인
	-n: 일련번호 출력

9. cp : 파일이나 디렉토리 복사
	-r : 하위디렉토리까지 모두 복사
	-b: 복사될 파일이름이 존재할 때 백업파일을 생성(덮어써지면서 백업파일 생성)
	-f: 이미 있는 파일명일 때 강제로 덮어쓰기
	-i : 이미 있는 파일명일 떄 덮어쓰기 여부를 묻는다.

10. vi 에디터: 문서 편집
	:w  >> 저장
	:q or :q! >> 종료
	:wq >> 저장 후 종료
	i or a  >>편집 모드
	esc 키 >> 명령모드
	(명령모드)
	dd >>  한줄 삭제(잘라내기)
	p >> 붙여넣기
	:set nu >> 라인번호(취소 :set nonu)
	/문자열 >> 해당 문자열로 이동
	:숫자 >> 해당 라인으로 이동
	:0	>> 줄의 맨앞
	:$	>> 줄의 맨뒤
